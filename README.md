# Firts steps in Robot Framework

## Util commands
- Inlcude Tags : `-e exmapleTag`
- Particular test case : `-t testCase`
- Test Suite (or folder with tests cases) : `-s Folder`

### Save reports

`robot -r ./Report/report.html -o ./Report/output.xml -l ./Report/logs.html`
